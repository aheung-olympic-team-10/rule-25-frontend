## Getting Started

### Yarn 설치

Yarn 이 설치되지 않은 환경에서는 아래 명령을 통해 설치합니다.

```
npm install -g yarn
```

### 의존성 설치

아래 명령을 통해 의존성 패키지를 설치합니다.

```
yarn
```

### 개발 서버 시작

아래 명령을 통해 개발 서버를 실행합니다. 개발 서버는 `localhost:3000` 으로 접속할 수 있습니다.

```
yarn start
```
